template<typename FeatType, unsigned int FeatDim>
int* TreeNode< FeatType, FeatDim >::m_leftChild

Index of the left child for the current node. The member is also used to define the type of node (intermediate, leaf or uninitialized) and can take one of the following values:

  • -1 if the node is a leaf node (i.e. it has no children);
  • -2 if the node is uninitialized;
  • >0 if the node is an intermediate node, i.e. it has children. The right children index node is given by the value m_leftChild+1.
