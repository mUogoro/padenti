Sample image data. Pixels data is stored continuously in data pointer, whereas labels stores the corresponding pixels labels. Both image channels (i.e. data parameter) and labels images must be of size width X height. Sampled pixels indices are stored within the vector pointed by samples parameter. The number of sampled pixels is returned.
